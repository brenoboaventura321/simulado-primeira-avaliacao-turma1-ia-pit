📝 Simulado Interativo: Algoritmos e Programação em C
Este projeto consiste em uma plataforma de avaliação interativa desenvolvida para estudantes de tecnologia. O sistema apresenta questões teóricas baseadas na linguagem C e lógica de programação, fornecendo feedback instantâneo e uma análise estatística de desempenho ao final.

🚀 Funcionalidades
Feedback em Tempo Real: Ao selecionar uma alternativa, o aluno descobre imediatamente se acertou ou errou, acompanhado de uma explicação técnica.

Interface Responsiva: Desenvolvido com Bootstrap 5, o simulado adapta-se a computadores, tablets e smartphones.

Resumo de Desempenho: Ao clicar em "Concluir", o sistema contabiliza acertos, erros e questões não respondidas.

Gráfico de Seção: Visualização dinâmica do aproveitamento através de um gráfico de pizza gerado com Chart.js.

Conteúdo Abrangente: Cobre desde conceitos básicos de algoritmos até estruturas condicionais complexas e formatação de dados em C.

🛠️ Tecnologias Utilizadas
HTML5: Estruturação do conteúdo.

CSS3 (Bootstrap 5): Estilização e layout responsivo.

JavaScript (ES6): Lógica de correção, manipulação do DOM e processamento de dados.

Chart.js: Biblioteca para geração dos gráficos estatísticos.

📂 Como Executar o Projeto
Faça o download ou copie o código do arquivo .html.

Salve-o em sua máquina como simulado.html.

Abra o arquivo em qualquer navegador moderno (Chrome, Firefox, Edge ou Safari).

Nota: É necessário estar conectado à internet para que os links (CDN) do Bootstrap e do Chart.js funcionem corretamente.

📝 Estrutura da Prova
O simulado é composto por:

20 Questões de Múltipla Escolha: Focadas em sintaxe, lógica e tipos de dados.

Feedback Pedagógico: Cada questão possui uma explanation (explicação) que reforça o aprendizado durante a execução.

🤝 Contribuições
Este é um projeto educacional. Sinta-se à vontade para:

Adicionar novas questões ao array quizData no JavaScript.

Personalizar as cores e estilos no bloco <style>.

Implementar novas métricas de pontuação.
